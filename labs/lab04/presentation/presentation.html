Элементы презентации

Цели и задачи

Целью данной работы является получение навыков работы с репозиториями и менеджерами пакетов.

Выполнение лабораторной работы

Списки репозиториев

Выведите на экран список репозиториев: - dnf repolist Выведите на экран список пакетов, в названии или описании которых есть слово user: - dnf search user

Установка и удаление nmap

Установите nmap, предварительно изучив информацию по имеющимся пакетам: - dnf install nmap - dnf install nmap* Удаление: - dnf remove nmap - dnf remove nmap*

Установка и удаление группы пакетов RPM Development Tools

Установка: - dnf groups info “RPM Development Tools” - dnf groupinstall “RPM Development Tools” Удаление: - dnf groupremove “RPM Development Tools”

dnf history

  • dnf history

Установка lynx

  • dnf list lynx
  • dnf install lynx –downloadonly

Файлы lynx

Используя rpm, определите по имени файла, к какому пакету принадлежит lynx, и получите дополнительную информацию о содержимом пакета: - rpm -qf $(which lynx) - rpm -qi lynx

Содержание скриптов и удаление пакета

Выведите на экран расположение и содержание скриптов, выполняемых при установке пакета: - rpm -q –scripts lynx Вернитесь в терминал с учётной записью root и удалите пакет: - rpm -e lynx

Пакет dnsmasq

Все действия аналогичны с действиями с предыдущим пакетом.
